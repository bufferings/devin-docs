# ecspresso

ecspresso（エスプレッソと発音）は、Amazon ECS（Elastic Container Service）のためのデプロイツールです。AWS ECSリソースを簡単に管理するための機能を提供します。

## 主な機能

- ECSサービスとタスク定義の管理
- 既存のECSサービスからの設定ファイル生成
- サービスのデプロイと更新
- Blue/Greenデプロイメント（AWS CodeDeployとの統合）
- タスクの実行と管理
- 複数環境での設定管理
- テンプレート機能によるパラメータ化
